Setting up Anki for Pi PAO Practice

Which system should I build for?

Century (2-2-2)Millennium (3-2-3)
Deck size~300~2100
Digits per chunk68
Effort to set upLowHigh

Recommendation: start with Century. You can add Millennium later — the app supports both simultaneously.

Step 1 — Create a note type in Anki

Open Anki. Tools → Manage Note Types → Add. Choose "Add: Basic" and name it PAO. Click Fields and set up: Number, Person, Action, Object, PersonImg, ObjectImg.

Step 2 — Set up card templates

In the note type editor, click Cards. Create three card types:

  • Card 1: Front: {{PersonImg}} → Back: {{Person}}<br>{{FrontSide}}<hr>{{Number}}
  • Card 2: Front: {{ObjectImg}} → Back: {{Object}}<br>{{FrontSide}}<hr>{{Number}}
  • Card 3: Front: {{Action}} → Back: {{FrontSide}}<hr>{{Number}}

Step 3 — Import your PAO list into Anki

Open sample-pao-system.xlsx and go to the Century Import or Millennium Import sheet. File → Save a Copy → CSV UTF-8 (.csv). In Anki: File → Import, set Note Type to PAO, map columns: 1 → Number, 2 → Person, 3 → Action, 4 → Object.

Step 4 — Add images to your cards

In the Anki Card Browser, click into the PersonImg or ObjectImg field and paste an image or use the editor toolbar to upload.

Step 5 — Export the .txt file

In Anki: File → Export. Choose "Notes in Plain Text (.txt)", select your deck, uncheck "Include tags". The output has columns: Number, Person, Action, Object, PersonImg, ObjectImg. Load this file in the app under Settings → Anki Images.

Step 6 — Load media into the app

Chrome/Edge: use the folder picker to point to your Anki collection.media. Other browsers: zip the folder and upload the .zip. Images are stored locally in IndexedDB and persist across reloads.
